# Privacy Policy
**Effective Date:** November 25, 2025  
**Last Updated:** November 25, 2025

## Overview
PMOMax PID Architect ("we", "us", "our") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our project management document processing service.

## Data We Collect
### Information You Provide
- Project documents and text you upload or paste
- Chat queries and instructions
- Exported documents (generated locally)

### Automatically Collected Information
- Usage metrics (documents parsed, queries made, exports generated)
- Performance metrics (response times, error rates)
- Browser information and IP address (for rate limiting)

## How We Use Your Data
### Primary Purpose
- Process project documents to generate Project Initiation Documents (PIDs)
- Provide AI-powered chat assistance for project management
- Generate exports (PDF, DOCX) of your project data

### Analytics & Improvement
- Monitor system performance and reliability
- Track usage patterns to improve service quality
- Aggregate anonymous usage statistics

## Data Storage & Retention
### Session-Only Processing
- **We do not persistently store your project data**
- All document text and PID data exists only in your browser session
- Server processing is transient (memory only, no database)
- Data is automatically cleared when you close your browser

### Temporary Server Processing
- Documents are processed in server memory during API calls
- Processing typically completes in 2-10 seconds
- No server-side caching of document content
- Server logs may contain request metadata (timestamps, sizes) but not document content

### Usage Metrics Retention
- Aggregated usage counts retained for billing purposes
- No personally identifiable information linked to usage metrics
- Metrics retained for 90 days for compliance reporting

## Third-Party Services
### Google Cloud Platform
- Application hosted on Google Cloud Run
- API keys stored in Google Secret Manager
- Subject to [Google Cloud Privacy Policy](https://cloud.google.com/terms/cloud-privacy-notice)

### Gemini AI API
- Document text sent to Google's Gemini AI for processing
- Subject to [Google AI Terms of Service](https://ai.google.dev/terms)
- Google may use data to improve AI models per their policies

## Data Security
### In Transit
- All connections use HTTPS/TLS encryption
- API keys transmitted securely via environment variables

### At Rest
- No persistent storage of project documents
- Session data stored only in browser memory
- Usage metrics stored in encrypted Cloud environment

### Security Measures
- Input validation and sanitization
- Rate limiting to prevent abuse
- Security headers (CSP, X-Frame-Options, etc.)
- Regular security audits and updates

## Your Rights
### Access & Control
- You control all project data in your browser
- No account or login required
- Clear data by closing browser tab

### Data Portability
- Export your PID as PDF or DOCX at any time
- No vendor lock-in - standard formats

### Deletion
- Close browser tab to immediately delete session data
- Server-side processing data expires within seconds
- Contact us to request deletion of usage metrics

## Cookies & Tracking
- We do not use cookies for tracking
- No third-party advertising or analytics cookies
- Session state maintained in browser memory only

## Children's Privacy
This service is not intended for users under 13 years old. We do not knowingly collect information from children.

## International Users

### CCPA Compliance
