# PMOMax User Manual (Updated)

## Introduction
PMOMax is a next-generation Project Initiation Document (PID) builder and analyzer. This manual reflects all recent UI, workflow, and documentation changes as of December 2025.

## Getting Started
- **Paste or drop files**: The main input area now supports drag-and-drop for .docx, .pdf, .txt, .json, .xlsx, .md, .csv files. Large text in the area says: "Paste or drop files here."
- **Browse & Upload**: The left panel features a large button with an up arrow and explicit file types. Text: "Drop files, or paste text above, or click here."
- **Supported file types**: .docx, .pdf, .txt, .json, .xlsx, .md, .csv
- **Clear**: The Clear button resets all app state, including input, assistants, notes, and project data.

## Main Features
- **AI Assistants**: Both left panel and Create Mode assistants have full state access and gold-themed UI.
- **Load Demo**: Minimal, single-line layout in the middle panel. Instruction left, button right.
- **Help & User Guide**: Updated for all new workflows and layouts.

## Step-by-Step
1. **Start a new project**: Click Clear, then +Create.
2. **Upload or drop a file**: Use the Browse button or drag into the paste area.
3. **Paste text**: Paste directly into the input area.
4. **Parse**: Click Parse to extract PID data.
5. **Edit and use AI**: Use assistants for suggestions and edits.
6. **Export**: Download as Word, PDF, or JSON.

## Best Practices
- Use Clear for a fresh start.
- Use demo data to explore features.
- Always check the Help for up-to-date guidance.
