# PMOMax User Guide (Short, Updated)

## Quick Start
- **Paste or drop files**: Drag-and-drop or paste .docx, .pdf, .txt, .json, .xlsx, .md, .csv into the main input area.
- **Browse & Upload**: Use the up arrow button with file types listed. Text: "Drop files, or paste text above, or click here."
- **Clear**: Resets all app state.

## Key Features
- Gold-themed, unified UI
- Smart AI assistants (full state access)
- Minimal Load Demo layout
- Updated Help & User Guide

## Best Practices
- Use Clear for a fresh start
- Use demo data to explore
- Check Help for guidance
