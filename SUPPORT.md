2. **Browser/OS:** Chrome 120 / macOS 14, etc.

4. **Screenshots:** Visual evidence of problem
5. **Document Size:** Page count and file size (for PDF issues)

## Common Issues & Solutions
### PDF Parsing Fails
**Symptoms:** Upload completes but no PID generated  
**Causes:**
- File too large (>500 pages not supported)
- Corrupted PDF
- Password-protected PDF
- Scanned images without OCR

**Solutions:**
1. Try smaller documents (<100 pages recommended)
2. Use PDF repair tool: https://smallpdf.com/repair-pdf
3. Remove password protection
4. OCR scanned documents before upload

### Chat Assistant Not Responding
**Symptoms:** Spinner spins indefinitely  
**Causes:**
- Network connectivity issue
- Server overload
- API rate limit exceeded

**Solutions:**
1. Refresh browser tab
2. Check internet connection
3. Wait 5 minutes and retry (rate limit resets)
4. Clear browser cache

### Gantt Chart Not Displaying
**Symptoms:** Timeline tab shows error or blank  
**Causes:**
- Missing dates in deliverables
- Invalid date format
- No deliverables defined

**Solutions:**
1. Ensure all deliverables have start and end dates
2. Use YYYY-MM-DD format
3. Add at least one deliverable

### Export Fails
**Symptoms:** Download doesn't start or file is corrupted  
**Causes:**
- Browser popup blocker
- Browser download limit
- Large document size

**Solutions:**
1. Allow popups from pmomax.ai
2. Check browser download settings
3. Try exporting individual sections
